# زياد خالد محمد سعد — Portfolio

موقع بورتفوليو شخصي لـ **زياد خالد محمد سعد** — مطور ويب وطالب في كلية الحاسبات والمعلومات، جامعة بنها، مصر.

## 🌐 رابط الموقع

[https://zyadkhaled-zk.github.io/portfolio/](https://zyadkhaled-zk.github.io/portfolio/)

## ✨ مميزات الموقع

- تصميم داكن احترافي بألوان ذهبية وزرقاء-خضراء (Gold & Teal)
- دعم كامل للغة العربية (RTL)
- متجاوب مع جميع أحجام الشاشات (Responsive)
- تأثيرات حركية للنجوم في الخلفية
- مؤشر مخصص (Custom Cursor) — يعمل على سطح المكتب فقط
- قسم للمشاريع مع تصفية (Filter) حسب النوع
- نموذج تواصل وظيفي عبر البريد الإلكتروني
- تأثير الكتابة الآلي (Typewriter) في العنوان

## 🛠️ التقنيات المستخدمة

| التقنية | الاستخدام |
|---------|-----------|
| HTML5 | هيكل الصفحة + إمكانية الوصول (Accessibility) |
| CSS3 | التصميم + الرسوم المتحركة + التجاوب |
| JavaScript (ES6+) | التفاعلية + Canvas + IntersectionObserver |
| Google Fonts | خطا Tajawal و Amiri |
| GitHub Pages | الاستضافة |

## 📁 هيكل المشروع

```
portfolio/

├── _config.yml         # إعدادات GitHub Pages
└── projects/

    ├── color-picker.html     # منتقي الألوان

```

## 🚀 تشغيل الموقع محلياً

```bash
# استنساخ المستودع
git clone https://github.com/ZyadKhaled-ZK/portfolio.git
cd portfolio

# تشغيل خادم محلي (مثال بـ Python)
python -m http.server 8000
# ثم افتح http://localhost:8000 في المتصفح
```
